# vue init
> 方便开发的vue初始化，基于vue3

## Project setup 初始化项目
```
npm install
```

### Compiles and hot-reloads for development 启动项目
```
npm run serve
```

### Compiles and minifies for production 构建
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
